# Login Use Case

The login use case is already implemented.

## Use Case Analysis

### Use Case Description

- **Actor:** unauthenticated user
- **Action:** login
- **Steps**
  - ask user to type the password
  - authenticate user
- **Alternate flows**
  - If password is not correct
    - display an error message
    - return to main menu
